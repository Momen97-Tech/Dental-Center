# 🦷 Dental Clinic Management System

A simple web-based application to manage patient records for a dental clinic using **HTML, CSS, and JavaScript**.

## 🚀 Features

* ➕ Add new patient records
* ✏️ Update existing records
* 🗑️ Delete single or all records
* 🔍 Search by:

  * Name
  * Diagnosis
  * Treatment
  * Date
* 💰 Automatic remaining cost calculation
* 💾 Data stored using LocalStorage (no backend needed)
* 🔊 Sound effects for actions

## 🛠️ Technologies Used

* HTML5
* CSS3 (Glassmorphism UI + Responsive Design)
* JavaScript (Vanilla JS)
* LocalStorage API
